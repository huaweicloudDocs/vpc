# 错误码<a name="ZH-CN_TOPIC_0020313978"></a>

## 功能说明<a name="section25523120121746"></a>

API调用发生错误时，会有错误结构体返回，该小节主要是对VPC封装接口（不包括OpenStack原生接口）错误结构的解释。

## 返回体格式<a name="section17199210121746"></a>

```
{
    "code": "VPC.0001",
    "message": "Token is null."
}
```
