# 删除端口<a name="ZH-CN_TOPIC_0201534173"></a>

## 功能介绍<a name="section17444113011524"></a>

删除端口。

接口约束：

device\_owner不为空的端口不允许删除。

## URI<a name="section44451130155215"></a>

DELETE /v1/\{project\_id\}/ports/\{port\_id\}
