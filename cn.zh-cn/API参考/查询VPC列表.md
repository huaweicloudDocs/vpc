# 查询VPC列表<a name="vpc_api01_0003"></a>

## 功能介绍<a name="section14477792"></a>

## URI<a name="section63191266"></a>

样例：

```
GET https://{Endpoint}/v1/{project_id}/vpcs?limit=10&marker=13551d6b-755d-4757-b956-536f674975c0
```
