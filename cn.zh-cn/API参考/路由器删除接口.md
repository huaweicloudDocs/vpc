# 路由器删除接口<a name="ZH-CN_TOPIC_0201534251"></a>

## 功能介绍<a name="section30197552205849"></a>

删除路由器接口。

接口约束：

如果该子网上还有负载均衡器对象，不允许删除路由器接口。

## URI<a name="section2308632205849"></a>

PUT /v2.0/routers/\{router\_id\}/remove\_router\_interface
