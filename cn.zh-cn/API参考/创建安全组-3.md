## 请求示例<a name="section577571911618"></a>

-   指定名称和描述创建安全组

    ```
    "POST https://{Endpoint}/v3/{project_id}/vpc/security-groups"
    
    {
        "security_group": {
            "name": "security_group_1", 
            "description": "security group description"
        }
    }
    ```

-   指定dry\_run预检创建安全组

    ```
    "POST https://{Endpoint}/v3/{project_id}/vpc/security-groups"
    
    {
        "security_group": {
            "name": "security_group_1", 
            "description": "security group description"
        }, 
        "dry_run": true
    }
    
    ```
